### 문제 설명

<p>ある貧乏な国のおてんばで勇敢なお姫様が，政略結婚のため別の国に嫁ぐことになった．ところがお姫様を亡き者としようとしている悪漢が嫁ぎ先への道の途中で刺客を放っている．</p>

<p>お姫様を無事に相手国に送り届けるため，あなたは安全な経路を既に決定していたのだが，お姫様の今までに通ったことのない道を通ってみたいという<s>わがままな</s>たっての願いで別の道を通ることとなった．そこであなたは地図を見ながらお姫様が通る道を決めなおすことにした．</p>

<p>全ての道は，宿場同士をつなぐ街道である．便宜上，出発地点及び目的地点も宿場とする．ところが，新しい道は治安の問題を抱えていた．盗賊やお姫様を亡き者にしようとする刺客が襲いかかってくる可能性が高いのである．</p>

<p>そのような危険な道を通るには護衛を雇うことが望ましい．護衛は宿場で雇うことができ，道単位で姫を守らせることができる．護衛が守っている間は盗賊や刺客に襲われることはないが，距離 1 につき金 1 がかかる．そのため，護衛を雇うためには所持金よりも次の宿場までの距離が長くないことが条件となる．</p>

<p>いま，与えられた予算 <i>L</i> のもとで，姫が無事に目的地に着くまでに襲いかかってくる盗賊や刺客の人数を最小化することを考える．あなたの仕事は，その最小化された人数を求めることである．なお，宿場にいる間に襲われることはないものとする．</p>

### 입력 

 <p>入力は複数のデータセットからなる．各データセットは次の形式をしている．</p>

<pre><i>N</i> <i>M</i> <i>L</i>
<i>A</i><sub>1</sub> <i>B</i><sub>1</sub> <i>D</i><sub>1</sub> <i>E</i><sub>1</sub>
<i>A</i><sub>2</sub> <i>B</i><sub>2</sub> <i>D</i><sub>2</sub> <i>E</i><sub>2</sub>
...
<i>A</i><sub><i>N</i></sub> <i>B</i><sub><i>N</i></sub> <i>D</i><sub><i>N</i></sub> <i>E</i><sub><i>N</i></sub>
</pre>

<p>最初の行には 3 つの非負の整数 <i>N</i> (2 ≤ <i>N</i> ≤ 100)，<i>M</i>， <i>L</i> (0 ≤ <i>L</i> ≤ 100) が与えられる．これらの整数は，宿場の数，道の数，護衛を雇うための予算を表す．宿場には 1 から <i>N</i> までの番号が割り振られており，出発地には 1 ，目的地には <i>N</i> の番号がそれぞれ割り振られている．</p>

<p>続く <i>M</i> 行では道の情報が各行に 1 つずつ与えられる．道の情報は 4 つの整数 <i>A</i><sub><i>i</i></sub>，<i>B</i><sub><i>i</i></sub> (1 ≤ <i>A</i><sub><i>i</i></sub> < <i>B</i><sub><i>i</i></sub> ≤ N)， <i>D</i><sub><i>i</i></sub> (1 ≤ <i>D</i><sub><i>i</i></sub> ≤ 100) <i>E</i><sub><i>i</i></sub> (0 ≤ <i>E</i><sub><i>i</i></sub> ≤ 10000) で与えられる．これらはそれぞれ，道の始点と終点の宿場の番号，道の距離，盗賊や刺客に襲われる人数を表す．</p>

<p>道は双方向に通行可能であり，かつある宿場の組に対しては高々 1 つの道しか存在しない．また，出発地から目的地には必ず移動可能であることが保証されている．</p>

<p>入力の終わりは，空白で区切られた 3 つの 0 を含む 1 行で示される．</p>

### 출력 

 <p>各データセットについて，盗賊や刺客に襲われる人数の最小値を各行に出力せよ．出力に余計な空白や改行を含めてはならない．</p>
